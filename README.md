# bb
野球の打者指標を算出する
![baseball_logo](sport_baseball_bat.png)

## Description
このソフトウェアでは野球の打者の打席数や安打数などの基本成績から打率やOPSなどの指標を算出する．

## Usage

```sh
bb [OPTIONS] <FILE>
OPTIONS
    -s, --sort            OPSの高い順に並べ替える
    -h, --help            このメッセージを出力する
ARGUMENTS
    FILE                  打者の基本成績がまとめられたcsvファイル
```
